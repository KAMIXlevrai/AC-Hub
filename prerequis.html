PRÉREQUIS

Avant de télécharger les mods, il est important de vérifier que vous avez installé les logiciels et les frameworks nécessaires pour leurs bon fonctionnements.

✔️ LES BONS LOGICIELS À UTILISER ✔️

qBittorrent

Client torrent open-source, léger et intuitif avec fonctionnalités avancées comme planification, cryptage et prise en charge des flux RSS.

Télécharger

Winrar

Logiciel de compression/décompression prenant en charge de nombreux formats de fichiers avec interface utilisateur conviviale et fonctionnalités avancées telles que création de fichiers auto-extractibles, protection par mot de passe et réparation de fichiers endommagés.

Télécharger

JDownloader 2

Gestionnaire de téléchargement open-source pour nombreux hébergeurs de fichiers avec interface utilisateur intuitive et fonctionnalités avancées telles que prise en charge de liens directs, accélération de vitesse et planification de téléchargements, permettant de télécharger en parallèle.

Télécharger

❗️ LES FRAMEWORKS À AVOIR OBLIGATOIREMENT ❗️

DirectX

Collection de technologies développées par Microsoft pour améliorer les performances graphiques et audio sur Windows.

Télécharger

Microsoft Visual C++

Ensemble d'outils de développement pour créer des applications C++ pour Windows et d'autres plateformes.

Télécharger

VcRedist

Package de redistribution de Microsoft Visual C++ pour installer les bibliothèques de runtime requises pour exécuter les applications développées avec Visual C++ sur un ordinateur.
